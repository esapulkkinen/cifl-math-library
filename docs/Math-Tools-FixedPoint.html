      Functor f => Visitor (Rec f) #

      Note the builder and visitor for Rec f allows every level of the recursion to be done by different function. This allows distinctions by level to be implemented easily as nested RecFold data structures. Otherwise this distinction would have to be included in the function itself (using fmap).
