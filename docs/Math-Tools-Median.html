Rules expected of MedianAlgebra.

From TAOCP section 7.1.1 http://www-cs-faculty.stanford.edu/~uno/taocp.html

"median/majority"

med x x y == x

"median/commutative"

med x y z == med x z y == med y z x == med z x y == med z y x

"median/associative"

med x w (med y w z) == med (med x w y) w z

"median/distributive"

med (med x y z) u v == med x (med y u v) (med z u v)
